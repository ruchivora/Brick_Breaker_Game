## :checkered_flag: Brick Breaker Game 
- **The game is made using java Swing**

## Instructions To Run The Game 
- **1. Clone the Repository**
- **2. Compile the following three files**
  - 1. Game.java
  - 2. GamePlay.java
  - 3. MapGenerator.java
- **3. Run Game.java file**
